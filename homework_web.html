About the project

The project was designed to understand the number of crimes commited in US where a gun was involved. Our team, has analized some variables like the number of guns with license per state, education and income to try to understand the increasing number of incidents from 2014 to 2017.

The data was obtained from dataworld and Census, we have done some data cleaning with Pandas and the visualizations were created with Seaborn library and matplotlib.
